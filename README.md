## 프로젝트 소개 : Ncurses를 이용한 파일 관리자
### 터미널을 이용하여 파일들을 관리하기가 힘들어서 생각해내게되었습니다.
Help

r = rename file

m = move file

d = delete file

c =copy file

enter = select
